# Demo Images

Use this folder for demo account assets.

Suggested files:

- `hook-<topic>.png` for `hook-composite` slides.
- `cta-<offer>.jpg` for `cta` slides.
- `screenshot-<source>.png` for `screenshot-slide` or `content-image` slides.
- `reference-<name>.jpg` for visual references.

Use repo-relative paths such as `./input/images/demo/hook-example.png`.
