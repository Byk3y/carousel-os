# create-carousel-os

Scaffold a local Carousel OS workspace for generating branded social carousels with Claude Code, Codex, Cursor, Antigravity, or any coding agent.

```bash
npm create carousel-os@latest my-content-system
cd my-content-system
npm install
cp .env.example .env
npm run init
```

The generated project includes:

- brand setup with `npm run init`
- local 1080x1350 PNG rendering
- `AGENTS.md`, `CLAUDE.md`, Cursor rules, and a project Claude Skill
- optional Postiz discovery and publishing scripts

Main repo: https://github.com/Byk3y/carousel-os
